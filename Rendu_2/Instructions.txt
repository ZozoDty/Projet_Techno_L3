L'application :
-------------

Vous devez développer une application unique contenant les effets listés dans les feuilles de TD.
L'interface n'est pas notée mais doit permettre d'appeler tous les effets implémentés et de réinitialiser une image. Les images utilisées seront dans les ressources. Fournissez des images qui permettent de mettre en évidence les effets (images de faible contraste pour les algorithmes de rehaussement de contraste, etc).
On demande un code commenté et lisible (nommage soigné, pas de duplication sur le code java).

dépôt git - vérifier qu'un checkout de votre projet dans un nouveau répertoire peut être importé correctement dans android studio et être compilé.
